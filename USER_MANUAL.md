# 需求分析系统用户手册

## 系统简介

需求分析系统是一个集需求收集、分析、评估和管理于一体的综合性工具。该系统支持多种需求分析方法，包括KANO模型、价值流图(VSM)、SMART目标设定、WFMT动作时间分析等，帮助项目团队更好地理解、分类和优先排序需求。

## 系统功能

### 1. 项目管理
- 创建和管理项目
- 查看项目详细信息
- 删除不需要的项目

### 2. 干系人管理
- 添加和管理项目干系人
- 记录干系人的角色、影响力和关注度
- 分析干系人对需求的关注点

### 3. 需求收集
- 收集需求的九要素信息：
  - 场景描述
  - 解决的问题
  - 当前解决方案
  - 目标
  - 预期解决方案
  - 价值
  - 其他信息
- 设置需求类型、来源、优先级等属性

### 4. 需求分析
- KANO模型分析：将需求分类为基本型、期望型、兴奋型、无差异型和反向型
- 价值流图(VSM)分析：分析需求的周期时间和交付时间
- SMART目标设定：确保需求目标明确、可衡量、可达成、相关性强、有时限
- WFMT动作时间分析：评估需求实现的时间成本和改善潜力

### 5. 价值评估
- 评估需求的商业价值、用户价值和技术价值
- 计算需求的投资回报率(ROI)
- 跟踪需求的实际价值实现情况

### 6. 路线图规划
- 制定项目需求实现的路线图
- 设置里程碑和截止日期
- 跟踪需求实现进度

### 7. 看板视图
- 以看板形式展示需求状态
- 支持拖拽操作更新需求状态

### 8. 综合分析
- 提供项目的综合分析报告
- 展示各项分析结果的统计数据
- 生成需求分析建议

### 9. 数据导入导出
- 支持CSV格式的数据导入导出
- 提供多种分析模板下载
- 支持PDF文档的需求提取

## 使用指南

### 系统登录
1. 启动系统后，在浏览器中访问 `http://localhost:5000`
2. 使用默认账号登录：
   - 用户名：admin
   - 密码：admin123
3. 建议登录后立即修改默认密码

### 创建项目
1. 在首页点击"创建项目"按钮
2. 输入项目名称和描述
3. 点击"创建"完成项目创建

### 管理干系人
1. 进入项目详情页
2. 点击"干系人管理"选项卡
3. 点击"添加干系人"按钮
4. 填写干系人信息，包括姓名、角色、影响力(1-5)、关注度(1-5)等
5. 点击"保存"完成添加

### 收集需求
1. 在项目详情页点击"需求采集"选项卡
2. 点击"添加需求"按钮
3. 填写需求的九要素信息及其他属性
4. 点击"保存"完成需求添加

### 进行需求分析
系统支持多种分析方法：

#### KANO分析
1. 在项目详情页点击"KANO分析"选项卡
2. 根据KANO模型对需求进行分类
3. 系统会自动计算需求优先级分数

#### VSM分析
1. 在项目详情页点击"VSM分析"选项卡
2. 填写需求的流程步骤、周期时间和交付时间
3. 系统会计算流程效率

#### SMART目标设定
1. 在项目详情页点击"SMART目标"选项卡
2. 为需求设置明确、可衡量、可达成、相关性强、有时限的目标

#### WFMT分析
1. 在项目详情页点击"WFMT分析"选项卡
2. 填写需求的标准时间、改善潜力等信息

### 评估需求价值
1. 在项目详情页点击"价值评估"选项卡
2. 评估需求的商业价值、用户价值和技术价值(1-10分)
3. 估算需求的实现难度(1-10分)
4. 系统会自动计算预估的投资回报率(ROI)

### 制定路线图
1. 在项目详情页点击"路线图规划"选项卡
2. 点击"添加里程碑"按钮
3. 设置里程碑的标题、描述和截止日期
4. 将需求分配到相应的里程碑中

### 查看综合分析报告
1. 在项目详情页点击"综合分析"选项卡
2. 系统会展示各项分析的统计数据
3. 根据分析结果生成需求管理建议

### 导入导出数据
1. 在项目详情页点击"导入导出"选项卡
2. 在"数据导入"区域选择要上传的CSV文件
3. 在"数据导出"区域选择要导出的数据类型和格式

## 常见问题

### 如何修改密码？
系统当前版本暂不支持用户自行修改密码，如需修改请联系系统管理员。

### 如何备份数据？
系统数据存储在SQLite数据库文件中，可直接备份instance目录下的database.db文件。

### 导入PDF文件失败怎么办？
请确保PDF文件是文本型PDF而非图片型PDF，系统只能提取文本型PDF中的内容。
